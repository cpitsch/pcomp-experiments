- Event logs from Ostovar et al. Robust Drift Characterization from Event Streams of Business Processes[^ostovar] ([Source](https://apromore.com/research-lab/#heading16))

- Each event log has 3000 cases with case IDs 0-2999

    - (0,999), (1000, 1999), (2000, 2999)
